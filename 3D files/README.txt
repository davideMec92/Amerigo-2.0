# Summary

A multi-purpose robot base/ chassis in order to prototype your robotics project.
Features:
Made out off 3mm acrylic(laser cut)
Easy to mount geared motors, arduino / arduino mega / raspberry pi
Easy to assemble
You can mount sensors and your micro-controller with ease 
 
Check out my instructable to make one: http://www.instructables.com/id/DIY-Multi-Purpose-Robot-Base-and-Motor-Shield/
 
To open the .svg file you can use Inkscape and Corel draw for .cdr
